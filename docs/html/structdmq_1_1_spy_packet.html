Detailed Description

Standardized packet containing bus traffic metadata.

This struct is passed to DataBus::Monitor subscribers and is designed to be serialized for transmission to external diagnostic tools.

The timestamp_us field uses dmq::Clock::now(), which typically provides monotonic time since boot.
