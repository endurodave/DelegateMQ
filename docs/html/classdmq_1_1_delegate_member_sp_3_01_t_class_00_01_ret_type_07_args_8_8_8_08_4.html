Detailed Description

template<class TClass, class RetType, class... Args>
class dmq::DelegateMemberSp< TClass, RetType(Args...)>

DelegateMemberSp<> class synchronously invokes a class member target function using a weak pointer.

This class is a "safe" version of DelegateMember. It holds a std::weak_ptr to the target. If the target is destroyed, the callback is silently dropped.

Template Parameters
TClassThe class type that contains the member function.
RetTypeThe return type of the bound delegate function.
ArgsThe argument types of the bound delegate function.
