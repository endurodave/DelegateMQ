Detailed Description

template<class RetType, class... Args>
class dmq::ISerializer< RetType(Args...)>

Delegate serializer interface for serializing and deserializing remote delegate arguments. Implemented by application code if remote delegates are used.

All argument data is serialized into a stream. write() is called by the sender when the delegate is invoked. read() is called by the receiver upon reception of the remote message data bytes.
