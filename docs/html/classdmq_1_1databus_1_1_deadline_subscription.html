Construct a deadline-monitored DataBus subscription.

Parameters
topicDataBus topic to subscribe to.
deadlineMaximum allowed interval between deliveries. Must be > 0.
handlerCalled on each data delivery.
onMissedCalled when no delivery arrives within the deadline window.
threadOptional worker thread for both callbacks. If nullptr, handler fires on the publisher's thread and onMissed fires on the Timer::ProcessTimers() thread.
