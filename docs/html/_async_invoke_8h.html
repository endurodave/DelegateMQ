Detailed Description

Helper functions to simplify invoking a free or member function on the user-specified thread of control using a single line of code. Particually useful for creating asynhronous APIs (see AsyncAPI.cpp).

size_t send_data(const std::string& data) { return AsyncInvoke(send_data_private, comm_thread, WAIT_INFINITE, data); }

template<class Func , class... Args>
auto AsyncInvoke ( Func func,
Thread & thread,
const std::chrono::milliseconds & timeout,
Args &&... args )

Helper function to simplify asynchronous invoke of a free function. If this function is called by thread the function is invoked synchronously. Otherwise an asynchronous delegate is created to invoke the funciton on thread and block waiting for the target function call to complete before returning.

Parameters
[in]func- the function to invoke
[in]thread- the thread to invoke func
[in]timeout- the time to wait for invoke to complete
[in]args- the function argument(s) passed to func
