Detailed Description

template<class RetType, class... Args>
class dmq::Signal< RetType(Args...)>

Thread-safe multicast delegate returning RAII connection handles.

May be instantiated on the stack, as a class member, or on the heap. Signal<Sig> is the single signal type in the library.

inlinenodiscard

Subscribe a delegate and return a RAII connection handle.

The returned ScopedConnection automatically disconnects on scope exit. Safe to call regardless of how the Signal was allocated.

Returns
A ScopedConnection. Let it go out of scope to auto-disconnect, or call Disconnect() manually.
