Detailed Description

Default DelegateMQ library configuration.

To override, define DMQ_USER_CONFIG as a path to your own config header: -DDMQ_USER_CONFIG="path/to/DelegateMQConfig.h" Your config only needs to define the values you want to change. Unset values fall back to these defaults via DelegateMQConfig_Default.h.
