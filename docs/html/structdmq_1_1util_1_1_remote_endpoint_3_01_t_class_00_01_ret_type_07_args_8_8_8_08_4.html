Detailed Description

template<class TClass, class RetType, class... Args>
struct dmq::util::RemoteEndpoint< TClass, RetType(Args...)>

RemoteEndpoint wraps a DelegateMemberRemote and its associated transport infrastructure into a single, cohesive unit.

This structure is designed to simplify the management of remote endpoints by grouping the delegate, the transport, and the necessary stream for serialization.

Template Parameters
TClassThe class type that contains the member function.
RetTypeThe return type of the member function.
ArgsThe types of the function arguments.
