Create a timer-safe async delegate for a non-const member function.

Returns a callable that can be connected directly to Timer::OnExpired. At most one dispatch is in flight at any time; excess timer ticks are dropped rather than queued. DelegateAsyncWait is structurally prevented — the dispatch is always non-blocking.

Parameters
objRaw pointer to the target object. Must outlive the delegate.
methodvoid() non-const member function to invoke.
thrThread on which method is dispatched. Must use FullPolicy::FAULT or FullPolicy::DROP, not TIMEOUT (see Timer.h).
onStuckOptional. Called from ProcessTimers() when stuck is detected.
stuckTimeoutDuration after which a non-expiring in-flight token triggers onStuck. Zero disables stuck detection.
m_timerConn = m_timer.OnExpired.Connect(
MakeTimerDelegate(this, &MyClass::OnTick, m_thread));
// With stuck detection:
m_timerConn = m_timer.OnExpired.Connect(
MakeTimerDelegate(this, &MyClass::OnTick, m_thread,
[this]{ OnThreadStuck(); }, std::chrono::seconds(2)));
